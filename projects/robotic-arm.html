Personal Project

4-DOF Robotic Servo Arm

A continually developed, modular robotic manipulator platform for pick-and-place and manipulation tasks, featuring interchangeable end-effectors.

Jun 2022 - Present Robotics
Robotic Arm Fusion 360 Arduino Iterative Design

Project Overview

This ongoing project represents a long-term development of a versatile robotic manipulator platform. The 4-degree-of-freedom arm is designed for educational use, research prototyping, and practical automation tasks.

System Specifications

  • Degrees of Freedom: 4 (Base rotation, shoulder, elbow, wrist rotation)
  • Control System: Arduino Mega 2560 with custom control software
  • Actuation: Digital servos (20kg·cm for base/shoulder, 10kg·cm for elbow/wrist)
  • Construction: 3D-printed PETG structural components, aluminum mounting hardware
  • Workspace: 300mm reach, 500g payload capacity

Gripper Development & Iteration

  • Version 1: Parallel-jaw gripper with direct servo actuation. Suffered from servo stalling under load.
  • Version 2: Four-bar linkage gripper with mechanical advantage. Reduced servo load but introduced complexity.
  • Version 3 (Current): Passive closing mechanism with spring return. Eliminates servo stalling completely, provides adaptive grasping.

Key Engineering Features

  • Modular End-Effectors: Quick-change mechanism for swapping between gripper, suction cup, and custom tooling
  • Forward Kinematics: Implemented in control software for Cartesian position control
  • Teach & Playback: Manual positioning mode to record and replay motion sequences
  • Safety Features: Software torque limiting, mechanical hard stops, emergency stop input

Applications & Demonstrations

  • Pick-and-Place: Successfully demonstrated sorting objects by color and size
  • Educational Tool: Used in 3D Printing Club workshops to teach robotic fundamentals
  • Research Platform: Serves as a testbed for control algorithm development and sensor integration
  • Future Development: Computer vision integration, force sensing, mobile base mounting
