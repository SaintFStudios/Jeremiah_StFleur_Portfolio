Personal Project

S.C.A.N. Drone (Servo-Controlled App Navigator)

A custom-designed terrestrial drone using continuous-rotation servos for differential steering, controlled via a mobile application with multiple operational modes.

Mar 2025 - Present Robotics & Controls
Robotics ESP32 PID Control Mobile UI

Project Overview

The S.C.A.N. Drone is a terrestrial mobile platform designed for modular payload testing and educational demonstrations. The project emphasizes precise control, wireless communication, and system integration.

Technical Specifications

  • Microcontroller: ESP32 with BLE/Wi-Fi capabilities
  • Actuation: Continuous-rotation servos with positional feedback
  • Control Interface: RemoteXY mobile application with three operational modes
  • Sensors: Ultrasonic distance sensor, IMU (planned)
  • Communication: Bluetooth Low Energy (BLE) for real-time control

Engineering Challenges & Solutions

  • Brownout Issues: Consolidated all electronics onto a single PCB with proper power regulation, eliminating voltage drops during servo actuation.
  • Precision Control: Implemented PID control with positional feedback for accurate differential steering (±2° accuracy).
  • System Integration: Designed a modular chassis that accommodates various sensor payloads while maintaining a compact footprint.

Results & Performance

  • Reduced system footprint by 20% through PCB consolidation
  • Achieved 95% reliability in wireless communication at 30m range
  • Implemented three operational modes: Reconnaissance, Communications Relay, and Decoy
  • Successfully demonstrated payload interchangeability with 5+ sensor configurations
